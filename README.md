# blcokchain
```
宗旨:
2009年起虛擬貨幣市場崛起(如Bitcoin)，區塊鏈新經濟時代來臨，
國際研究暨顧問機構 Gartner更預測 2019 年企業必須了解的十大策略性科技趨勢中
即包含了區塊鏈的領域。本課程帶領學員了解區塊鏈底層技術，
並讓學員實作區塊鏈中制訂合約時所使用的特殊協議-智能合約(smart contract)。
講師將帶領學員撰寫智能合約，並探討智能合約之安全威脅與實作，
課程更以Blockchain CTF為實戰場域並透過解題來強化學員實戰能力。


課程內容:
1.Blockchain: A historical journal
2.Blockchain實務:智能合約(Smart Contract)
3.智能合約安全
4.Blockchain CTF實戰
```
